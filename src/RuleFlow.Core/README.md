# RuleFlow.Core

Default implementation of the RuleFlow engine: **`RuleEngine`**, fluent **`Rule<T>`** / **`RuleSet<T>`** builders, **`RuleContext`**, formatters, and the persistence mapper/registry.

## Contents

- **Engine** — synchronous and asynchronous evaluation, execution options, explainability pipeline
- **Rules** — fluent rule and rule set construction
- **Formatting** — e.g. `TextTreeFormatter`, JSON formatters
- **Persistence** — `RuleRegistry<T>`, `RuleDefinitionMapper<T>` for JSON definitions
